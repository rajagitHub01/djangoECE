About This Project

This is a Django-based Product Management System designed to handle user authentication and complete product operations including adding, searching, updating, and deleting products efficiently.

User Authentication

Secure user registration and login system using Django backend, ensuring protected access to product management features.

Product Management

Perform full CRUD operations — Create, Read, Update, and Delete — with an intuitive and responsive user interface.

Search Functionality

Advanced search system that allows filtering products by name or brand using Django ORM queries.

Modern UI Design

Built using Bootstrap 5 with responsive layout, professional styling, and clean UX principles.

Developed as a full-stack learning project to demonstrate backend logic, database handling, and UI integration using Django.
